# Greater Macedonia Mod

Custom Hearts of Iron IV mod content for a Greater Macedonia alternate-history campaign.

## Repository Contents

- `common/`: country setup, ideas, decisions, focus tree, unit equipment, and script enums.
- `events/`: narrative and gameplay events.
- `history/`: country, state, and unit history definitions.
- `localisation/english/`: English localisation text.
- `gfx/`: flags, portraits, ideas, and interface-related textures.
- `interface/`: GFX declarations and UI hooks.
- `map/`: map-side support data.
- `descriptor.mod`: HOI4 mod descriptor.

## Requirements

- Hearts of Iron IV
- A compatible game version matching this mod's script/content format

## Development Notes

- Keep localisation keys synchronized with scripted content.
- Validate scripted files after major focus/event changes.
- Preserve HOI4 text encodings and formatting conventions when editing game files.
